# Airbnb Data Analysis (Tableau)

## Overview
This project analyzes Airbnb listing data to identify trends in pricing, location, and revenue. The goal is to understand factors that influence pricing and booking patterns.

## Tools Used
- Tableau
- Excel (for data preparation)

## Key Features
- Interactive dashboard analyzing pricing by number of bedrooms
- Geographic visualization of price distribution by zipcode
- Time series analysis of revenue trends
- Comparative analysis of pricing across different locations

## Key Insights
- Prices increase significantly as the number of bedrooms increases
- Certain zip codes have consistently higher average prices
- Revenue shows an overall upward trend throughout the year
- Most listings are concentrated in lower-bedroom categories

## Files
- `Airbnb_Dashboard.twbx` – Tableau workbook
- `dataset.xlsx` – Source data

## How to Use
1. Download the Tableau workbook
2. Open using Tableau Public or Tableau Desktop
3. Interact with filters and visualizations
